# 迪拜仓库存优先发货系统 — 运行指南

> 📦 无需安装任何东西，setup.bat 会自动帮你搞定一切。

---

## 📋 电脑要求

- Windows 10 / 11
- 能正常上网
- 至少 5GB 空闲空间

---

## 第〇步：获取代码

1. 浏览器打开 **https://github.com/boluo0208/Noon**
2. 点击绿色 **`<> Code`** 按钮 → **`Download ZIP`**
3. 解压到桌面或其他位置

---

## 第一步：双击安装

在解压后的文件夹里找到：

```
dubai_shipping_tool → scripts → setup.bat
```

**双击 `setup.bat`**，它会：

- ✅ 检测并**自动安装 Python**（如果没有的话）
- ✅ 检测并**自动安装 Node.js**（如果没有的话）
- ✅ 配置 Python 虚拟环境 + 安装依赖
- ✅ 安装 Playwright 浏览器驱动
- ✅ 安装前端依赖

> ⚠️ 如果 Python 或 Node.js 是首次自动安装，脚本会提示你**重新打开一次** setup.bat，这是正常操作，照做就行。

整个过程 5-10 分钟。

---

## 第二步：双击启动

在同一个 scripts 文件夹里找到：

```
dubai_shipping_tool → scripts → start.bat
```

**双击 `start.bat`**，浏览器会自动打开系统页面。

---

## 🛑 停止

关掉弹出的命令行窗口即可。

---

## 🔧 常见问题

| 问题 | 解决 |
|------|------|
| winget 自动安装失败 | 手动下载：[Python](https://www.python.org/downloads/) + [Node.js](https://nodejs.org)，装好后重新运行 setup.bat |
| 页面打不开 | 等几秒，或手动访问 http://localhost:5173 |
| Noon 加载慢 | 国内访问 Noon 较慢，正常现象，耐心等待 |
| Export 按钮找不到 | Noon 页面可能更新了，截图反馈 |
