# 迪拜仓库存优先发货系统 — 运行指南

> 📦 本指南适用于 **没有安装任何开发环境的电脑**，从头配置并运行。

---

## 📋 电脑要求

- Windows 10 / 11
- 能正常上网
- 至少 5GB 空闲空间

---

## 第〇步：获取代码

### 方式 A：下载 ZIP（推荐，无需安装任何工具）

1. 浏览器打开 **https://github.com/boluo0208/Noon**
2. 点击绿色 **`<> Code`** 按钮 → **`Download ZIP`**
3. 解压到桌面或其他位置

### 方式 B：Git Clone（需要先安装 Git）

```bash
git clone https://github.com/boluo0208/Noon.git
```

---

## 第一步：安装 Python

1. 打开 **https://www.python.org/downloads/**
2. 点击黄色 **Download Python**（3.11 以上）
3. 双击安装包
4. ⚠️ **务必勾选底部的 `Add Python to PATH`**
5. 点击 `Install Now`，等待完成

**验证：** `Win+R` → `cmd` → 输入 `python --version`，显示版本号即成功。

---

## 第二步：安装 Node.js

1. 打开 **https://nodejs.org**
2. 下载 **LTS 版本**（左侧绿色按钮）
3. 双击安装，全部默认，一路 Next

**验证：** 命令行输入 `node --version`，显示版本号即成功。

---

## 第三步：Edge 浏览器

Windows 10/11 自带，无需额外安装。

---

## 第四步：一键安装依赖

找到文件夹：

```
dubai_shipping_tool → scripts → setup.bat
```

**双击 `setup.bat`**，等待 5-10 分钟自动安装。

---

## 第五步：一键启动

找到文件夹：

```
dubai_shipping_tool → scripts → start.bat
```

**双击 `start.bat`**，浏览器自动打开 `http://localhost:5173`。

---

## 🛑 停止

关掉弹出的两个命令行窗口。

---

## 🔧 常见问题

| 问题 | 解决 |
|------|------|
| `python not found` | 重装 Python，勾选 Add Python to PATH |
| `npm not found` | 重装 Node.js LTS |
| 页面打不开 | 等几秒，或手动访问 http://localhost:5173 |
| Noon 加载慢 | 国内访问 Noon 较慢，耐心等待 |
| Export 按钮找不到 | Noon 页面可能更新，截图反馈 |
